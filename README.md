# how to use 
![screen-shot](screen-shot.png)
## Dependencies
- electron
- electron-builder
  
## install packages
first install the packages!

``` bash
npm i --save
```

## build

``` bash
npm run build
```
and **wait** build complete.

## run as Administrator
to set DNS Addresses you should **run the program** by **admin**.

- go to **/dist/win-unpacked**

- right-click on **dns-changer.exe** and select **Propeties**

- from tabs select **Compatibility** and check **(Run this program as an administrator)**

- click **Apply** then **OK**


enjoy !
